Wow! Welcome onboard!

You are all set to use SCORA for your assessment needs!

As a first step, you can start adding Users for Roles such as Author, Test Administrator, Examiner and Participant. Each Role comes with a set of privileges and permissions which can be specifically assigned based on the task a User is expected to perform. While creating a User, in addition to the generic Fields, you can also create custom Fields. These SIX custom Fields can only be created or edited by you. Once Users are added, you can start inviting them to SCORA. Users can also be bulk imported into the system using the Excel template provided under the Data Templates Menu.

Once you have added Users, you can also Create and Manage User Groups as per your needs.

We would love to assist you with this process. Email us at cx@scora.io and let us help you to make the most of SCORA.
