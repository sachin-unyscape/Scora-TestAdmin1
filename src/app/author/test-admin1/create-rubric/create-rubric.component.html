SCORA allows you to create your own customised Rubric for a question Item that requires manual scoring (Free Text: Descriptive and Fill-in and Upload Item Types) using a table. Use the Rows to create as many criteria as you need and columns to define proficiency levels under each criterion. Define the Points to be awarded for each level, moving from lowest level in the first column to the highest level in the last column. Click on ‘+’ and ‘-‘ icons to add or delete rows and columns, as required.

SCORA encourages reusability of Rubrics to save you time. You can clone Rubrics or create different versions by making minor changes to an existing Rubric and use it for a different Item or for the same Item in a different context/Test. Add unique Keywords for each Rubric that will help users search and retrieve relevant Rubrics.
