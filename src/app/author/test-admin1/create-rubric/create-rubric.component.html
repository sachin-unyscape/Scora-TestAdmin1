Instructions

A Rubric is a framework for grading constructed responses such as descriptive answers, fill-in and assignments. Rubrics are used to establish a standard guideline when multiple examiners are grading responses for the same question Item to ensure objectivity and consistency.

SCORA allows you to create your own customised Rubric for a question Item that requires manual scoring (Free Text: Descriptive and Fill-in and Upload Item Types) using a table. Use the Rows to create as many criteria as you need and columns to define proficiency levels under each criterion. Define the Points to be awarded for each level, moving from lowest level in the first column to the highest level in the last column. Click on ‘+’ and ‘-‘ icons to add or delete rows and columns, as required. Add unique Keywords for each Rubric that will help users search and retrieve relevant Rubrics.

Points values can either be positive whole numbers or multiples of 0.25 or 0.33. Also, you cannot enter multiples of 0.25 and multiples of 0.33 as values for different cells in the same row.
